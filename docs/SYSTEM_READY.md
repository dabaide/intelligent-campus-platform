## 故障排除

### 问题: 登录失败

**解决方案**:
1. 确认演示账户已初始化
2. 检查用户名和密码
3. 查看浏览器控制台错误
4. 检查后端日志

### 问题: 聊天无回复

**解决方案**:
1. 检查后端服务是否运行
2. 验证DeepSeek API密钥
3. 查看后端日志
4. 检查网络连接

### 问题: 搜索无结果

**解决方案**:
1. 检查知识库是否已导入
2. 验证搜索词是否正确
3. 检查数据库连接
4. 查看系统日志

### 问题: 导出失败

**解决方案**:
1. 检查用户权限
2. 验证数据库连接
3. 检查磁盘空间
4. 查看系统日志
