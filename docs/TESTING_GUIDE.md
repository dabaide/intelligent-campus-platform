# 系统测试指南

## 概述

本指南提供了完整的系统测试步骤，包括功能测试、性能测试和集成测试。

---

## 前置准备

### 1. 启动服务

#### 启动后端服务
```bash
cd backend
# 激活虚拟环境
venv\Scripts\activate.ps1

# 启动uvicorn服务器
uvicorn main:app --reload --host 127.0.0.1 --port 8000
```

#### 启动前端服务
```bash
cd frontend
npm start
```

### 2. 初始化演示数据

```bash
cd backend
python scripts/init_demo_users.py
python scripts/import_hbue_knowledge.py
```

### 3. 访问应用

- **前端**: http://localhost:3000
- **后端API**: http://localhost:8000
- **API文档**: http://localhost:8000/api/v1/docs

## 性能测试

### 1. 响应时间测试

**测试工具**: curl 或 Postman

```bash
# 测试登录API响应时间
curl -X POST http://localhost:8000/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"student@example.com","password":"password123"}' \
  -w "\nTotal time: %{time_total}s\n"
```

**预期结果**:
- ✅ 响应时间 < 500ms
- ✅ 成功率 > 99%

### 2. 并发测试

**测试工具**: Apache Bench 或 wrk

```bash
# 测试并发请求
ab -n 100 -c 10 http://localhost:8000/health
```

**预期结果**:
- ✅ 处理100个请求，并发10个
- ✅ 成功率 > 99%
- ✅ 平均响应时间 < 100ms

**测试步骤**:

**预期结果**:

## 常见问题排查

### 问题1: 登录失败

**排查步骤**:
1. 检查演示账户是否已初始化
2. 验证用户名和密码
3. 检查后端服务是否运行
4. 查看浏览器控制台错误

### 问题2: 聊天无回复

**排查步骤**:
1. 检查后端服务日志
2. 验证DeepSeek API密钥
3. 检查网络连接
4. 查看API文档

### 问题3: 搜索无结果

**排查步骤**:
1. 检查知识库是否已导入
2. 验证搜索词是否正确
3. 检查数据库连接
4. 查看系统日志

### 问题4: 导出失败

**排查步骤**:
1. 检查用户权限
2. 验证数据库连接
3. 检查磁盘空间
4. 查看系统日志

## 7. 知识库管理测试

### 7.1 文档管理测试

**创建文档**:
```bash
curl -X POST http://localhost:8000/api/v1/knowledge-management/documents \
  -H "Authorization: Bearer <admin_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "测试文档",
    "content": "这是一个测试文档",
    "category": "academic",
    "tags": ["测试"],
    "is_public": true
  }'
```

**预期结果**:
- ✅ 文档成功创建
- ✅ 返回文档ID
- ✅ 缓存自动失效

**更新文档**:
```bash
curl -X PUT http://localhost:8000/api/v1/knowledge-management/documents/{document_id} \
  -H "Authorization: Bearer <admin_token>" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "更新后的标题",
    "content": "更新后的内容"
  }'
```

**列表文档**:
```bash
curl -X GET "http://localhost:8000/api/v1/knowledge-management/documents?category=academic&limit=20" \
  -H "Authorization: Bearer <admin_token>"
```
